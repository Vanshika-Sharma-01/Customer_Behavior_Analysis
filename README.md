# Customer Shopping Behavior Analysis

## Overview

This project focuses on analyzing customer shopping behavior to uncover patterns in purchasing, customer segments, and revenue trends. The workflow includes data loading, cleaning, exploratory data analysis (EDA), SQL-based querying, and building an interactive dashboard to visualize insights. The goal is to transform raw customer data into meaningful business insights that support data-driven decision making.

## Dataset

The dataset contains customer shopping information such as:

* Customer demographics (age, gender, location)
* Product details (item purchased, category, size, color)
* Transaction information (purchase amount, payment method, shipping type)
* Customer behavior indicators (subscription status, discounts, previous purchases, purchase frequency)
* Review ratings

This data helps analyze spending patterns, product popularity, and customer loyalty.

## Tools & Technologies

* **Python (Pandas, Jupyter Notebook)** – Data loading, cleaning, and EDA
* **MySQL** – Storing structured data and running analytical SQL queries
* **Power BI** – Building interactive dashboards and visualizing insights
* **Gamma** – Creating a presentation report for the project

## Project Steps

1. **Data Loading** – Imported the dataset using Python and Pandas.
2. **Data Cleaning** – Handled missing values, standardized column names, and prepared the dataset for analysis.
3. **Exploratory Data Analysis (EDA)** – Explored customer behavior, purchase patterns, and category trends.
4. **Database Integration** – Loaded the cleaned dataset into a MySQL database.
5. **SQL Analysis** – Executed SQL queries to answer business questions such as revenue trends, discount usage, and customer segmentation.
6. **Dashboard Creation** – Built a Power BI dashboard to visualize key metrics and insights.
7. **Reporting** – Summarized findings in a structured report and presentation.

## Dashboard

The Power BI dashboard highlights important business insights including:

* Total revenue and average purchase amount
* Customer distribution by gender and age group
* Product category performance
* Impact of discounts and subscription status on spending
* Customer segmentation and repeat purchase behavior

## Results

The analysis provided insights into customer purchasing patterns, high-performing product categories, and the impact of discounts and subscriptions on revenue. The dashboard enables easy exploration of key metrics and supports data-driven decision making.

## How to Run

1. Clone this repository.
2. Open the Python notebook to view data cleaning and EDA steps.
3. Import the dataset into MySQL and run the provided SQL queries.
4. Open the Power BI `.pbix` file to explore the interactive dashboard.
5. View the presentation/report for a summary of insights and findings.
